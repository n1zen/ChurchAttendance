# Church Attendance App
A church attendance app for [Fort Andres Bonifacio Adventist Church](https://fortandresbonifacio-sdachurch.org/).
# Features:
### Church members and visitors
- Sign up form for members and visitors
- Easy attendance logging
### Church admins
- Members list to show all existing members and visitors
- Member details list to review, edit, and delete member/visitor details
- Attendance list that shows present members/visitors filtered by date
- Data exporting as CSV or excle workbook
